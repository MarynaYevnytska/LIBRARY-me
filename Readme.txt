Даное приложение хранит информацию об авторах, книгах и читателях.
Вы можете добавить нового пользователя.
Выполнить проверку существующего пользователя.
Также при желании, можете познакомится с списком авторов
По библиграфичному номеру можно получить список всех книг выбранного автора
Есть статистическая информация, а именно:
читатель, который прочитал больше всех книг
автор, книги которого, пользуются наибольшей популярностью
